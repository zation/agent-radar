# v0.4 UI List Scroll and Spacing Fix Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修正导航、filter、列表溢出与间距，并把桌面 Tool/Evaluation 列表限制为 `min(60vh, 640px)` 内部滚动。

**Architecture:** 保留现有 AppShell、ToolsWorkspace、EvaluationPage 和 mobile drill-in。共享 Button/Toggle variants 负责交互状态，页面 Tailwind utilities 负责列表 max-height、overflow 与间距；不新增组件或依赖。

**Tech Stack:** React 19、TypeScript、shadcn/Base UI、Tailwind CSS 4、Node test runner、Vite。

## Global Constraints

- Desktop list 使用 `lg:max-h-[min(60vh,640px)] lg:overflow-y-auto`。
- Mobile 不限制 list 高度。
- 不修改 detail 滚动、定位或信息架构。
- 不新增 ScrollArea、分页、虚拟列表或依赖。
