# v0.9 P2 Local-First Skill Dataset Implementation Plan

- Status: Approved
- Implementation commits: None
- Current status source: [docs/15-roadmap.md](../../15-roadmap.md)

**Goal:** Replace the uncommitted online Skill adapter with a compatible, checksummed, atomically updated local dataset and local query workflow.

**Architecture:** Generate versioned Skill dataset releases during the existing artifact pipeline, inherit and revalidate earlier v1 releases from the previous reviewed bundle, and publish a mutable v1 channel pointer. The installed dependency-free Node client synchronizes into a versioned cache and performs all selection-data reads locally. The host agent reasons from local context under deterministic action ceilings.

## Constraints

- Do not call MCP or the hosted recommendation endpoint from the Skill.
- Do not add install hooks, background updates, credentials, new infrastructure, or paid services.
- Do not overwrite or silently mutate an inherited release directory.
- Do not switch `current.json` until all compatibility, size, checksum, and schema checks pass.
- Do not weaken existing installation, execution, permission, or human-confirmation boundaries.
