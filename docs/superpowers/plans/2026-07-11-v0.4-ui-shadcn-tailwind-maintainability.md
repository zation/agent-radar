**Goal:** 用 shadcn primitives 和 Tailwind utilities 重构当前 Tools / Evaluation UI，提高字号、统一英文 UI chrome，并删除页面层重复维护的原生控件行为和大段布局 CSS。

**Architecture:** 保留现有页面、状态纯函数和数据流。`src/components/ui/*` 提供 Button/Input/Textarea/Select/ToggleGroup/Progress 行为，feature 组件用 Tailwind 组合布局和领域状态；`styles.css` 收缩为 imports、tokens、基础 theme 与全局规则。用源码契约测试防止原生交互控件、超小字号和中文 UI chrome 回归。

**Tech Stack:** React 19、TypeScript 5.5、shadcn/Base UI、Tailwind CSS 4、class-variance-authority、Node test runner、Vite 8。

## Global Constraints

- 保留 Tools / Evaluation 信息架构、Recommend 状态机、候选排序、Evaluation 完整性判断和 Mobile drill-in。
- shadcn 已提供的 pointer、hover、active、focus-visible 不在页面层重复定义。
- Tailwind 负责 layout、spacing、typography、colors、responsive 和领域 selected/status variants。
- 不恢复默认 Card 网格，不给 facts、boundaries、evidence items 增加 Card 外壳。
- 可见 UI 最低 `text-xs`；表单、按钮、tool row 主文本最低 `text-sm`；正文使用 `text-sm`/`text-base`。
- UI chrome 和静态说明使用英文；Golden Query `query.task`、`review_notes` 和 artifact 内容保持原文。
- 不实现 OAuth、D1 vote、Issue Form、v0.5 英文化或新的 UI dependency。

## File Structure

- `tests/ui-maintainability.test.ts`：源码契约，锁定 shadcn primitives、字号与英文 chrome。
- `src/components/ui/button.tsx`：补充 Agent Radar 的 `nav`、`filter`、`row` variants 或尺寸，不改变 Base UI 行为。
- `src/ui/app-shell.tsx`：shadcn Button 导航 + Tailwind shell。
- `src/ui/tools-workspace.tsx`：shadcn form controls、filters、buttons 与 Tailwind workspace。
- `src/ui/tool-detail.tsx`：Tailwind 开放式详情布局与 shadcn Progress。
- `src/ui/evaluation-page.tsx`：shadcn filters/query rows 与 Tailwind summary/detail。
- `src/ui/styles.css`：只保留 imports、tokens、theme、body 与 reduced-motion。
- `docs/14-web-ui.md`、Roadmap、Spec、Plan：完成事实与提交范围。

```ts
import assert from "node:assert/strict";
import { readFile } from "node:fs/promises";
import test from "node:test";

const featureFiles = [
  "src/ui/app-shell.tsx",
  "src/ui/tools-workspace.tsx",
  "src/ui/evaluation-page.tsx"
];

test("feature UI uses shared shadcn primitives instead of native interactive tags", async () => {
  for (const path of featureFiles) {
    const source = await readFile(path, "utf8");
    assert.doesNotMatch(source, /<(button|input|textarea|select)\b/);
  }
});

test("visible UI does not use sub-12px font sizes", async () => {
  const css = await readFile("src/ui/styles.css", "utf8");
  assert.doesNotMatch(css, /font-size:\s*(?:[7-9]|10|11)px/);
});

test("UI chrome contains no hard-coded Chinese copy", async () => {
  for (const path of featureFiles) {
    const source = await readFile(path, "utf8");
    assert.doesNotMatch(source, /[\u3400-\u9fff]/);
  }
});
```
