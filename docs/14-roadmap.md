# 14 路线图

## 文档用途

本文件定义 Agent Radar 的阶段性目标和优先级，避免项目过早变成大而全平台。

## 应包含内容

- MVP：文档体系、Tool Card schema、少量采集源、静态索引、基础评分、MCP 查询。
- v0.2：更多来源、Web UI、golden queries、推荐解释、自动报告。
- v0.3：安全风险评分、用户反馈、评分回归、agent 自迭代任务。
- v1.0：稳定 API/MCP、可复现评测、可扩展插件、公开数据发布。
- 明确不做：早期账号系统、在线安装市场、复杂企业权限治理。

## 后续生成提示词

```text
请为 Agent Radar 生成路线图文档。请把项目拆成 MVP、v0.2、v0.3、v1.0，每个阶段说明目标、交付物、验收标准、风险和不做什么。要求优先验证“根据需求推荐合适 AI 工具”的核心价值，避免早期做过重平台。内容使用中文。
```

## 维护规则

- 路线图必须服务产品目标，不按技术兴趣扩张。
- 每个阶段都应有可演示、可评测的结果。
