# 09 推荐引擎

## 文档用途

本文件定义 Agent Radar 如何根据用户需求或 coding agent 任务推荐工具。推荐必须可解释，并能说明不推荐的理由。

## 应包含内容

- 输入：自然语言任务、技术栈、运行环境、风险偏好、已有工具、预算。
- 检索：关键词、分类过滤、语义搜索、来源过滤。
- 排序：任务匹配、评分、可信度、风险、维护状态、安装成本。
- 输出：Top N 候选、推荐理由、风险、安装方式、替代方案、什么时候不要用。
- AI 使用方式：MCP tools、JSON 输出、Markdown 解释。

## 后续生成提示词

```text
请为 Agent Radar 生成推荐引擎文档。请说明如何把用户的开发需求映射为工具候选，如何结合分类、评分、风险、维护状态和安装复杂度排序，如何输出 AI 可用的推荐解释。要求包含输入 schema、输出 schema、排序规则、反推荐规则、示例查询和示例结果。内容使用中文。
```

## 维护规则

- 推荐理由必须引用具体字段或评分依据。
- 当没有合适工具时，应明确返回“暂无可靠推荐”，而不是强行推荐。
