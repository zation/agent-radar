# 00 产品简报

## 文档用途

本文件定义 Agent Radar 的产品定位、目标用户、核心价值、竞品边界和成功指标。它是所有需求、架构、评分和路线图决策的上游依据。

## 应包含内容

- 一句话定位。
- 目标用户：AI-first 开发者、coding agent 使用者、AI 平台团队、工具维护者。
- 核心场景：根据开发需求找到最合适的 AI Agent、Skill、MCP、CLI 或 Framework。
- 产品不做什么：不只是新闻摘要，不只是 awesome list，不做未经验证的自动安装平台。
- 与相邻产品的差异：MCP registry、awesome list、AI 生态日报、安全扫描器、企业工具目录。
- 成功指标：推荐准确率、工具覆盖数、数据新鲜度、评分解释质量、Agent 决策可用率。

```text
请为 Agent Radar 生成一份完整产品简报。Agent Radar 是面向人类开发者和 coding agent 的 AI 工具评级与推荐知识库，覆盖 AI Agent、Skill、MCP Server、CLI、Framework 和 Prompt/Rules。请说明产品定位、用户画像、核心问题、主要场景、竞品边界、差异化价值、MVP 范围、非目标范围和成功指标。内容使用中文，结构清晰，避免营销空话。
```

## 维护规则

- 当产品定位变化时，先更新本文件，再更新需求和路线图。
- 不要把实现细节写进本文件，除非实现选择会影响产品边界。
