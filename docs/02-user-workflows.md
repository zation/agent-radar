# 02 用户流程

## 文档用途

本文件描述人类用户、coding agent 和维护者如何使用 Agent Radar。它用于校准产品体验和 API 设计。

## 应包含内容

- 人类开发者流程：输入需求、查看推荐、比较工具、理解风险、选择复用方式。
- Coding agent 流程：读取任务、调用 MCP 查询、获取候选、解释选择、请求人类批准、集成工具。
- 维护者流程：新增采集源、审核工具卡片、调整评分规则、处理误判样例。
- 失败流程：没有合适工具、工具不可信、评分冲突、数据过期、安装失败。

## 维护规则

- 所有重要功能都应该能对应到至少一个用户流程。
- 如果流程中出现高风险动作，必须链接到安全文档。
