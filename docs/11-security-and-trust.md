# 11 安全与信任

## 文档用途

本文件定义 Agent Radar 如何识别和表达工具安全风险。由于 MCP、Skill 和 Agent 工具可能接触文件、浏览器、数据库、云服务和账号权限，本文件是核心文档。

## 应包含内容

- 风险类型：供应链风险、权限过大、恶意工具、prompt injection、secret 泄露、远程代码执行、数据外传。
- 信任来源：官方、知名组织、活跃社区、个人项目、未知来源。
- 权限建模：文件系统、网络、浏览器、邮件、数据库、云账号、支付、代码执行。
- 安全评分：风险等级、置信度、证据来源。
- Human approval：哪些操作必须人工确认。
- 不信任原则：未知工具默认不自动安装、不自动执行。

```text
请为 Agent Radar 生成安全与信任文档。请覆盖 MCP Server、Skill、Agent、CLI、Framework 的主要风险，包括权限、供应链、prompt injection、数据外传、secret 泄露和远程代码执行。请定义风险等级、信任等级、证据字段、人工审批规则和推荐时的安全解释模板。内容使用中文。
```

## 维护规则

- 安全风险字段宁可保守，不要为了推荐率降低风险等级。
- 与权限、安装、执行相关的自动化必须默认最小权限。
